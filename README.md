### Usage
於空白欄位輸入事項，點擊新增鍵可將該項目新增至待辦清單。

點擊 finish 鍵可將該項目移至已完成清單中，並記錄事項完成的時間。

已完成清單中可點擊 Delete 將該項目移除。





輸入/移除事項，並紀錄完成時間。將紀錄結果以清單形式呈現。

紀錄輸入之值，並使用.push列出清單。
